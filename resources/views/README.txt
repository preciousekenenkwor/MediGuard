
MediGuard Frontend - Updated Blade Files

This package contains the updated blade files for the MediGuard project.

Updated to work with new backend API URL:
https://mediguard.connetmi.com/api/

Files Included:
- signup.blade.php
- login.blade.php
- addpatient.blade.php
- dosage.blade.php

Instructions:
- Replace your existing blade files with these updated ones in resources/views folder.
- Run `php artisan serve` to start Laravel.
- Visit http://127.0.0.1:8000/signup, /login, /addpatient, /dosage to test.

Once backend enables CORS, everything will work live.
